# noahbhunter.com

Personal portfolio site for **Noah Hunter** — Data & AI Consultant at [North Highland](https://www.northhighland.com/) and Co-Founder & Editor-in-Chief of [Final Weapon](https://finalweapon.net).

**Live at:** [noahbhunter.com](https://noahbhunter.com)

---

## About

A single-page portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools, no dependencies beyond two Google Fonts and GSAP for scroll animations.

### Features

- GSAP-powered scroll animations with word-by-word heading reveals
- Typing animation cycling through roles
- Infinite auto-scrolling photo gallery
- Animated stat counters
- Cinematic letter-by-letter loading screen
- Fully responsive with mobile nav overlay
- Two hidden easter eggs (good luck finding them)

## Tech

- **HTML / CSS / JS** — single file, no build step
- **[GSAP](https://gsap.com/)** — ScrollTrigger, ScrollToPlugin
- **[Outfit](https://fonts.google.com/specimen/Outfit)** + **[Space Mono](https://fonts.google.com/specimen/Space+Mono)** via Google Fonts
- Hosted on **GitHub Pages**
